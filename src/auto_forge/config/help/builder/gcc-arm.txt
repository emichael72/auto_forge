Tool: arm-linux-gnueabihf-gcc

This is the GNU cross compiler for 32-bit ARM targets (hard-float ABI).
It's required to compile binaries for ARM embedded platforms or Linux ARM targets.

Minimum required version: GCC 10.0

Installation:

Fedora:
  sudo dnf install arm-none-eabi-gcc-cs  # or arm-linux-gnueabihf-gcc

Ubuntu/Debian:
  sudo apt install gcc-arm-linux-gnueabihf

Other:
  Download from Linaro or your embedded toolchain provider.

Tip: Ensure the tool is in your PATH, or set its path explicitly in the toolchain config.
